# Kth Largest Element in an Array (Leetcode 215)

[Leetcode 215](https://leetcode.com/problems/kth-largest-element-in-an-array/)


Most likely the interviewer doesn't want you to use built-in functions. Always ask;


## Using built-in functions

```javascript
/**
 * @param {number[]} nums
 * @param {number} k
 * @return {number}
 */
var findKthLargest = function(nums, k) {

    nums.sort((a,b) => a - b).reverse();

    return nums[k-1];
    
};
```
