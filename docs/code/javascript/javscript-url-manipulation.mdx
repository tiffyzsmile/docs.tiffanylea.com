# Get URL Tiers

```javascript
const pathname = "/tier1/tier2/tier3";
// var tiers = pathname.substring(1).split("/"); // $: ["tier1","tier2","tier3"]
var tiers = pathname.split("/"); // $: ["","tier1","tier2","tier3"]
console.log("tier 1: ", tiers[1]);
console.log("tier 2: ", tiers[2]);
console.log("tier 3: ", tiers[3]);
```
