# Docker

## Overview

Docker CLI: [Docker CLI](https://docs.docker.com/engine/reference/commandline/cli/) 

Docker Compose CLI: [Docker Compose CLI](https://docs.docker.com/compose/reference/)

## Common CLI

```sh
// list all containers
$ docker ps -all

// stop container in list
$ docker stop [CONTAINERID]

// remove unused data
$ docker system prune

// remove unused local volumes
$ docker volume prune
```
